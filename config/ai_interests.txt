# TrendRadar AI 兴趣描述文件
# 当前目标：保留更可能引发公共争议、情绪波动、跨平台扩散和持续讨论的涉华公共议题；
# 尽量过滤普通资讯、日常通知、常规科普、低信息量社会奇闻和无明显争议性的内容。
#
# 从上到下优先级递减；一条内容命中多个方向时，优先归入更靠前的主题。

下面是我当前重点关注的涉华公共议题与争议风险方向：

1. 国际涉华叙事与跨境争议
重点保留境外主流媒体、评论机构、智库、跨境平台上与中国直接相关的报道和评论，尤其是容易引发对立叙事、跨平台传播、政策争论、外交安全争议和国际舆论波动的内容。

2. 港澳台海与周边局势
重点保留港澳台、周边地区和区域安全局势中，与中国相关且容易引发持续讨论、情绪分化、误读扩散、市场波动或外媒集中报道的内容。

3. 宏观经济、地产债务与市场金融争议
重点保留与中国宏观经济、房地产、地方债、资本市场、汇率、外贸、供应链、能源和大宗商品相关，并可能引发市场预期变化、公众焦虑、争议解读或持续讨论的内容。

4. 公共安全、事故灾害与突发风险
重点保留重大事故、公共安全、极端案事件、自然灾害、公共卫生等内容，尤其是容易出现谣言、争议扩散、二次传播、治理讨论或情绪放大的事件。

5. 社会民生与公共服务争议
重点保留就业、养老、医疗、教育、住房、交通、食品安全、生态环保、消费维权、公共服务调整等议题中，能够引发明显争论、治理评价分化、社会情绪波动或跨平台传播的内容。

6. 科技治理、平台治理与新技术风险
重点保留生成式 AI、深度伪造、平台治理、网络安全、数据安全、关键基础设施安全、异常传播、自动化账号等议题中，与中国相关且容易形成风险叙事或治理争议的内容。

7. 历史、文化与价值争议
重点保留历史叙事、文化事件、教育文化、网络亚文化、身份认同、公共价值讨论中，能够引发明显争议、长期传播或群体分化的内容。

8. 结构性社会议题与长期风险信号
重点保留青年就业、老龄化、未成年人保护、家庭结构、劳动权益、城市治理、区域发展等长期议题中，具有持续公共影响、争议空间或结构性风险信号的内容。

# 标题质量与筛选要求
# 即使命中主题，也请优先过滤以下内容：
- 普通节假日安排、学校活动安排、日常政策通知、常规宣传稿
- 一般生活常识、教育小知识、普通科普、无争议的实用信息
- 单纯猎奇、社会奇闻、低信息量意外事件，且没有进一步公共讨论价值
- 纯营销、广告、导流、带货、平台运营内容
- 标题党、情绪化夸张但缺乏实质信息量的内容
- 二手搬运、重复聚合、无可靠来源的转述
- 与中国关联很弱、只是蹭热点但没有明确公共议题价值的内容
- 单纯娱乐八卦、明星花边、饭圈日常互撕，除非已明显上升为公共争议事件

# 争议优先原则
# 对内容分类和保留时，优先判断：
- 是否可能引发公共争议、明显分歧、情绪对立或舆论拉扯
- 是否存在跨平台扩散、持续发酵、外媒跟进、二次传播或议程升级潜力
- 是否会影响公共认知、政策讨论、市场预期、治理评价或国际涉华叙事
- 是否具有“风险信号”属性，而不是普通资讯属性

# 明确降权示例
# 以下类型即便短时上榜，也应尽量不保留：
- “奖状在小学教育中的作用有多大”
- “春假全市具体安排”
- 普通气球爆燃、小范围猎奇事件，若无持续争议或公共风险外溢
- 一般校园、社区、节庆、便民通知类热点
